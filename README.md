# CRM مغازه

سیستم مدیریت مشتری و وچر تخفیف، مخصوص فروشگاه.

## ساختار
- `index.html` — ورود کارمندان
- `dashboard.html` — لیست و جستجوی مشتری‌ها، افزودن مشتری جدید
- `customer.html` — پروفایل مشتری، ثبت خرید، نمایش وچرها
- `scan.html` — اسکن/وارد کردن کد وچر برای اعمال تخفیف
- `firestore.rules` — قوانین امنیتی دیتابیس (فقط کارمند لاگین‌شده دسترسی دارد)

## مراحل قبل از استفاده
1. این فولدر را در گیت‌هاب آپلود کن و GitHub Pages را فعال کن.
2. توی Firebase Console → Authentication → Users → یک یا چند حساب کارمند (ایمیل/رمز) دستی بساز.
3. توی Firebase Console → Firestore Database → Rules → محتوای `firestore.rules` را کپی و Publish کن.
4. سایت را باز کن، با همون ایمیل/رمز کارمند وارد شو.

## هنوز باقی مانده (مرحله بعد)
- وصل کردن EmailJS برای ارسال خودکار ایمیل وچر بهمشتری (فعلاً فقط در کنسول لاگ می‌شود).
